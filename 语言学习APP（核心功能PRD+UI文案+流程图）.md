# 语言学习APP（核心功能PRD+UI文案+流程图）

# 一、产品基础信息（PRD前置）

## 1.1 产品名称

暂定：瞬说（贴合“快速开口、即时反馈”核心，简洁好记，适配全年龄段语言学习者）

## 1.2 产品定位

零基础友好型语言学习APP，主打“无枯燥背诵、强场景落地、高娱乐互动”，以「图像闪记」「短剧配音」为核心功能，让用户在“刷、玩、演”中掌握语言，核心目标是“降低开口门槛，实现碎片化高效学习”。

## 1.3 目标用户

- 核心用户：18-35岁，零基础/入门级语言学习者（学生、职场人、出国备旅人），讨厌死记硬背，追求“好玩、高效、能直接用”的学习方式。

- 次要用户：有一定基础，想提升口语发音、积累日常地道表达的学习者，需要高频互动场景巩固所学。

## 1.4 核心功能范围

本次PRD仅聚焦两大核心功能：「单词不用背·图像闪记」「短剧配音·最上瘾」，包含功能逻辑、交互规则、异常处理，其余辅助功能（首页、个人中心等）仅做基础适配，不展开细节。

## 1.5 版本信息

MVP版本（V1.0）：仅上线两大核心功能，完成闭环体验；后续版本迭代补充社交、关卡、AI语伴等延伸功能。

# 二、核心功能PRD（可直接交付开发）

## 模块一：单词不用背·图像闪记

### 2.1.1 功能概述

以“图像+声音+短句”为核心，采用极简闪卡形式，结合间隔重复算法，让用户无需看中文释义、无需刻意背诵，通过场景画面与语言声音的绑定，形成条件反射式记忆，适配碎片化学习场景（通勤、排队等），全程无压力、高反馈。

### 2.1.2 核心需求

- 用户需求：摆脱枯燥背单词模式，快速记住高频实用单词，且能直接运用到句子中，不记拼写、不抠释义，重点掌握“发音+含义”。

- 产品需求：提升用户日活、单次使用时长，降低零基础用户入门门槛，实现“学完就能用”的闭环，为后续短剧配音功能提供词汇储备。

### 2.1.3 功能详细规则

#### 1. 闪卡基础规则

- 闪卡内容结构（固定顺序，全屏展示，无多余干扰）：
        

    1. 顶部：极简提示（仅显示“听发音→想画面”，无其他文字）；

    2. 中间：高清真实场景图（占屏幕80%，无卡通图，贴合日常场景，如“coffee”对应“咖啡店桌面咖啡杯”“冰咖啡”“手冲咖啡”等不同场景图）；

    3. 图下：目标语单词/短语（大号字体，加粗，仅目标语，无中文释义）；

    4. 底部：目标语短句（小号字体，贴合图片场景，如“coffee”对应“I drink coffee.”“I want a coffee.”，短句仅包含当前单词，无复杂词汇）；

    5. 隐藏元素：点击图片/单词/短句，播放母语者真人发音（清晰、自然，无机械感），可重复点击播放。

- 闪卡操作交互（竖屏全屏，单指操作，极简流畅）：
        

    1. 左滑：跳过当前闪卡（标记为“暂不记忆”，后续间隔1小时再次出现）；

    2. 右滑：记住当前闪卡（标记为“已初步记忆”，后续间隔3小时再次出现）；

    3. 上滑：收藏当前闪卡（加入“我的收藏”，可手动复习，不参与系统间隔推送）；

    4. 下滑：听慢读发音（当前单词+短句的慢速发音，适合零基础模仿，正常发音为默认播放，点击即播，下滑切换为慢读）；

    5. 长按图片：显示当前单词的极简词性（不显示中文，如“n. 名词”“v. 动词”），仅作辅助，不强制展示。

- 闪卡推送规则（根据用户操作和反应速度，动态调整，不设固定数量，避免用户疲劳）：
        

    1. 首次进入：推送5张基础高频词（如“hello”“hi”“eat”“drink”“go”），适应操作；

    2. 日常推送：系统根据用户“右滑正确率”“操作速度”自动调整每日闪卡数量（范围：10-50张），正确率≥80%，次日增加5张；正确率＜50%，次日减少5张；

    3. 间隔重复算法（核心，实现“不用背”）：
                

        - 首次右滑（初步记忆）：3小时后再次推送；

        - 第二次右滑（巩固记忆）：12小时后再次推送；

        - 第三次右滑（熟练记忆）：24小时后再次推送；

        - 第四次右滑（完全记忆）：72小时后推送一次，确认记忆后，不再主动推送，仅存入“已掌握单词库”；

        - 左滑（暂不记忆）：1小时后再次推送，若连续3次左滑，标记为“用户不常用”，暂停推送，1天后再次尝试推送。

#### 2. 特色子功能规则

##### （1）闭眼听词模式

- 入口：闪记页面顶部，点击“闭眼听”按钮（图标为“眼睛+喇叭”，简洁明显）；

- 功能逻辑：
        

    1. 进入模式后，屏幕自动变黑（仅显示极简进度条，无其他内容）；

    2. 系统自动播放单词发音（正常速度），每播放1个单词，停顿3秒，让用户在脑中联想对应画面；

    3. 停顿结束后，用户可操作：单指点击屏幕（想出来画面）→ 亮屏显示对应闪卡（验证联想是否正确），右滑/左滑操作不变；长按屏幕（没想出来）→ 直接亮屏显示闪卡，标记为“暂不记忆”；

    4. 退出模式：点击屏幕边缘“退出”按钮（浅色图标，不干扰），或连续5次长按屏幕，自动退出。

- 反馈：每次点击验证正确，播放“叮”的轻快音效；连续3次正确，弹出极简提示“大脑反应超快！”。

##### （2）0.5秒极速闪卡模式

- 入口：闪记页面顶部，点击“极速闪”按钮（图标为“闪电”）；

- 功能逻辑：
       

    1. 进入模式后，闪卡自动切换，每张闪卡仅停留0.5秒，自动播放单词发音（无短句发音，聚焦单词记忆）；

    2. 用户无需手动操作，系统自动记录“用户是否有停留操作”（若用户点击图片，视为“关注该单词”，后续间隔1小时再次推送）；

    3. 模式时长：默认1分钟（约120张闪卡），结束后弹出“今日极速闪记完成，记住了XX个高频词”的反馈；

    4. 退出模式：点击屏幕底部“暂停”按钮，即可退出，退出后保存当前进度，下次进入可继续。

##### （3）情绪单词分组

- 入口：闪记页面底部，点击“情绪分组”（图标为“笑脸”，展开后显示4个分组）；

- 分组分类（固定4组，覆盖日常高频情绪相关单词，后续可迭代增加）：
        

    1. 开心组：happy、smile、fun、delicious、exciting（图片均为笑脸、美食、欢乐场景，自带积极情绪）；

    2. 疲惫组：tired、sleepy、busy、boring（图片为揉眼睛、加班、睡觉场景）；

    3. 生气组：angry、rude、annoying（图片为皱眉、叉腰等轻微生气场景，不涉及负面过激画面）；

    4. 日常组：normal、okay、fine、common（图片为日常穿搭、通勤、居家场景）。

- 交互：选择对应分组后，仅推送该分组下的闪卡，操作规则与普通闪记一致，分组切换无延迟。

#### 3. 数据反馈规则

- 实时反馈（闪记操作后即时显示，停留2秒后自动消失，不干扰操作）：
        

    1. 右滑：“√ 记住啦！大脑又进步了”+ 轻快音效；

    2. 左滑：“→ 下次再试，不急～”；

    3. 上滑：“⭐ 已收藏，随时复习”；

    4. 连续5次右滑：“🎉 连续正确！你也太厉害啦”。

- 每日反馈（闪记页面底部，点击“我的进度”查看，简洁数据可视化，无复杂图表）：
        

    1. 核心数据：今日记住单词数、大脑反应速度（与昨日对比，如“比昨天快12%”）、已掌握单词总数；

    2. 预测提示：根据当前进度，给出极简预测，如“再坚持3天，就能流利说出点餐相关单词啦”；

    3. 数据保存：自动保存历史数据，支持查看近7天进度，无手动删除功能（避免用户误删，增强成就感）。

#### 4. 异常处理

- 无网络场景：进入闪记页面时，弹出提示“当前无网络，已为你加载离线闪卡（昨日未完成的闪卡）”，离线状态下仅可查看、听已缓存的发音，无法同步数据，网络恢复后自动同步；

- 发音播放失败：点击发音后，若播放失败，弹出提示“发音加载中，请稍候”，3秒后仍失败，自动切换为文字提示“可尝试下滑听慢读，或检查网络”；

- 操作无响应：连续3次操作无响应（如左滑无反应），弹出提示“轻轻滑动即可操作哦”，同时震动手机（轻微震动，不打扰）；

- 闪卡加载失败：页面显示“闪卡加载中...”，5秒后仍失败，弹出“刷新试试”按钮，点击后重新加载，多次加载失败，提示“请退出页面后重新进入”。

## 模块二：短剧配音·最上瘾

### 2.2.1 功能概述

以“15秒内小短剧”为载体，结合AI配音评分、对口型辅助、多角色演绎等功能，让用户在“扮演角色、录制配音”中练习口语发音、语调、节奏，生成可保存、可分享的成品视频，以娱乐性驱动高频使用，同时巩固图像闪记所学单词，实现“学用结合”。

### 2.2.2 核心需求

- 用户需求：摆脱枯燥跟读，在趣味演绎中提升口语，敢于开口，同时有成就感（成品可炫耀），无需专业水平，零基础也能参与。

- 产品需求：提升用户留存率、分享率，打造核心差异化优势，实现“图像闪记学单词→短剧配音用单词”的闭环，增强用户粘性。

### 2.2.3 功能详细规则

#### 1. 短剧库规则

- 短剧分类（固定4类，贴合日常场景，MVP版本每类上线10条，共40条，后续迭代新增）：
       

    1. 日常小剧场（核心，适配零基础）：点餐、问路、打招呼、买东西、打车，台词简单，多为短句，包含图像闪记中的高频单词；

    2. 情感小剧场：安慰、表白、道歉、吐槽、撒娇，台词带轻微情绪，适合练习语调；

    3. 搞笑小剧场：社死、误会、神回复、沙雕对话，台词轻松有趣，增强娱乐性，降低开口压力；

    4. 热血小剧场：鼓励、加油、自信发言，台词有力量，适合提升开口自信。

- 短剧基础规范（统一标准，保证用户体验，便于开发）：
        

    1. 时长：8-15秒，不超过15秒（避免用户疲劳，提升完成率）；

    2. 台词：每段短剧2-5句台词，单句长度不超过10个单词，无复杂语法、生僻单词，优先使用图像闪记中的高频词；

    3. 画面：2D动画画面（简洁清新，人物形象可爱，无复杂场景，重点突出人物对话），画面节奏与台词节奏同步；

    4. 配音：默认提供母语者标准配音（清晰、自然，带对应情绪），支持慢速、正常、连读三档播放；

    5. 难度标注：每条短剧标注难度星（1-3星），1星（零基础）、2星（入门）、3星（基础提升），方便用户选择。

- 短剧展示规则：
        

    1. 短剧列表：竖屏卡片式展示，每张卡片包含“封面图+标题+难度星+标签”，封面图为短剧核心画面，标题简洁有吸引力（如《便利店不小心说漏嘴》《第一次和外国人搭讪》）；

    2. 排序规则：默认按“推荐”排序（根据用户图像闪记所学单词，推送包含对应单词的短剧），支持按“难度”“分类”筛选，支持搜索（输入关键词，如“点餐”“表白”，查找对应短剧）；

    3. 更新规则：每日更新3条短剧（1条日常、1条搞笑、1条随机），标记为“今日新更”，吸引用户每日打开。

#### 2. 配音流程规则（核心，极致顺滑，降低操作门槛）

整体流程：选剧本 → 看原片 → 逐句配音 → 打分反馈 → 合成成片 → 保存/分享，共5步，每步操作简单，无多余步骤，全程有引导。

##### 步骤1：选剧本（入口清晰，引导明确）

- 操作：点击短剧卡片，进入短剧详情页，点击“开始配音”按钮（大号按钮，居中展示，颜色突出，如橙色，与页面其他元素区分）；

- 引导提示：首次进入配音，弹出极简引导“先看原片，再逐句录制，大胆开口就好～”，仅显示1次，后续不再弹出；

- 异常：若短剧未加载完成，点击“开始配音”后，显示“加载中...”，同时播放轻快背景音乐，缓解用户等待焦虑。

##### 步骤2：看原片（快速熟悉台词和节奏）

- 操作：进入原片播放页，自动播放标准配音版短剧（正常速度），画面下方显示逐字高亮字幕（目标语，无中文），字幕颜色与画面区分明显（如白色字幕，黑色阴影）；

- 辅助操作：
        

    1. 播放控制：支持暂停、重播，支持拖动进度条（精准到0.1秒）；

    2. 速度切换：页面底部有“慢速、正常、连读”三档按钮，点击可切换播放速度（零基础默认推荐“慢速”）；

    3. 单句循环：点击字幕，可单独播放该句配音，支持循环播放（适合零基础模仿）。

- 进入下一步：原片播放完成后，自动弹出“开始录制”按钮，点击即可进入逐句配音环节；也可在原片播放过程中，点击“跳过原片，直接录制”按钮（小号按钮，不突出，避免干扰）。

##### 步骤3：逐句配音（核心环节，高反馈、低压力）

- 界面布局：

    1. 顶部：当前录制句数（如“1/3”），清晰显示进度；

    2. 中间：短剧画面（暂停状态，显示当前台词对应的画面），画面下方显示当前台词（大号字体，目标语，无中文）；

    3. 底部：录制控制区（“录制”按钮、“重录”按钮、“跳过”按钮、速度切换按钮）。

- 录制规则：
        

    1. 点击“录制”按钮，开始录制，按钮变为“停止”，同时画面轻微动效（如边框闪烁），提示用户“正在录制”；

    2. 录制时长：单句录制时长不超过5秒，若超过5秒，自动停止录制，提示“语速慢一点哦，再试一次～”；

    3. 重录：录制完成后，点击“重录”按钮，可重新录制当前句子，无次数限制；

    4. 跳过：点击“跳过”按钮，可跳过当前句子（标记为“未录制”，后续可在合成前补充录制，若未补充，合成时使用默认配音）；

    5. 对口型辅助（核心细节）：录制时，画面下方显示“节奏条”（蓝色=正常，红色=太快，黄色=太慢），同步用户发音节奏，提示用户调整语速，贴合原片对口型；

    6. 背景音：录制时，可选择是否播放短剧背景音（默认不播放，避免干扰用户发音，页面底部有“背景音”开关按钮）。

##### 步骤4：打分反馈（即时反馈，增强成就感）

- 打分维度（4个维度，精准且易懂，不复杂）：发音（40分）、语调（20分）、重音（20分）、节奏（20分），总分100分；

- 反馈规则：
        

    1. 单句打分：每句录制完成后，1秒内给出打分，显示“当前句得分XX分”，同时标红错误部分（如发音错误的单词，标注“发音偏差”，并播放该单词的标准发音，提示用户模仿）；

    2. 情绪加分：若用户发音带对应情绪（如搞笑短剧带幽默语调、情感短剧带温柔语调），额外加5-10分，提示“情绪很到位！额外加分啦～”；

    3. 总分反馈：全部句子录制完成后，给出总分，同时显示“得分评价”（如80-100分：“太棒啦！发音超标准，堪比母语者～”；60-79分：“不错哦！再调整一下语调，会更完美～”；＜60分：“加油！多模仿几次，你一定可以的～”）；

    4. 错题记录：自动记录得分低于60分的句子，提示“这些句子可以再优化哦”，点击可重新录制。

##### 步骤5：合成成片（自动合成，无需手动操作）

- 合成规则：打分完成后，点击“合成成片”按钮，系统自动合成“用户配音+原片画面+逐字字幕”，合成时长不超过3秒；

- 成片效果：字幕同步用户配音节奏，画面与原片一致，结尾添加极简水印（“瞬说APP | 你的专属配音作品”），不影响观看体验；

- 配音皮肤（特色功能）：合成前，页面底部可选择“配音皮肤”（默认无皮肤，即用户原声），可选皮肤：御姐音、正太音、温柔音、搞笑电音，点击即可预览效果，选择后同步应用到全部句子，无额外操作；

- 异常：合成失败时，弹出提示“合成失败，请检查网络，重新尝试”，同时提供“重新合成”按钮，点击即可再次合成。

##### 步骤6：保存/分享（强化成就感，提升分享率）

- 操作按钮：成片播放完成后，页面底部显示“保存本地”“分享”两个大号按钮，居中展示，颜色突出；

- 保存规则：点击“保存本地”，自动保存到手机相册，同时弹出提示“已保存到相册，快去看看吧～”，保存格式为MP4（高清，不压缩画质）；
